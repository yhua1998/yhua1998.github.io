# Animation

由两部分组成

- 描述动画的样式规则
- 指定动画开始、结束、中间点样式的关键帧

## 配置动画

- `animation`简写模式

- `animation-delay`

- `animation-direction`

  - `normal`
  - `reverse`
  - `alternate`
  - `alternate-reverse`

- `animation-duration`

- `animation-iteration-count`

  - `number`
  - `infinite`

- `animation-name`

- `animation-play-state`

  - `running`
  - `paused`

- `animation-timing-function`

  - `linear`
  - `ease-in`
  - `ease-out`
  - `ease-in-out`
  - `ease`
  - `step-start`
  - `step-end`

  - `cubic-bezier`
  - `steps`
    - `end`
    - `start`
    - `jump-start/end/none/both`

- `animation-fill-mode`
  - `none`
  - `forwards`
  - `backwards`
  - `both`
