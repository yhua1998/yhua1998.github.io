# Flex 弹性盒子布局

- `flex-direction`

  - `row`
  - `column`
  - `row-reverse`
  - `column-reverse`

- `flex-wrap`
  - `wrap`

> 上述两个属性的简写: `flex-flow: row wrap`

- `flex`
  - `flex:1`: flex-grow
  - `flex:1 200px`: flex-grow flex-basis
  - `flex:1 1 200px`:flex-grow flex-shrink flex-basis

> `flex`是`flex-grow flex-shrink flex-basis`的简写


- `align-items`
    - stretch
    - center
    = flex-start/end

- `align-self`

- `justify-content`
    - flex-start/end
    - center
    - space-around
    - space-between

- `order`:排序
