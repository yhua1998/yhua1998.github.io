# 媒体查询

```css
@media media-type and (media-feature-rule) {
    /* css rules go here */
}
```

- media-type
    - all
    - print
    - screen
    - speech

- media-feature-rule
    - width
    - max-width
    - min-width
    - orientation
        - portrait:竖
        - landscape:横
    - hover

可以使用`and`对媒体特征进行混合

使用`,`表示或
